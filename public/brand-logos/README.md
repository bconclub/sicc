## Logo Specifications

- **Format**: PNG (with transparent background preferred) or SVG
- **Size**: Recommended 192x112 pixels (width:height ratio ~1.7:1)
- **Background**: Transparent or white
- **Style**: Brand standard colors and fonts
- **Quality**: High resolution for crisp display

## Usage

These logos are displayed in the BrandLogos component which shows:
- 3 rows of scrolling brand logos
- Infinite scroll animation (alternating directions)
- Grayscale effect with hover to color
- Used on Home page and Technical Specifications page
